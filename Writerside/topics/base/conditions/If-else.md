# If, else, else-if

### If

```base
if( condition ) {
    // TODO si condition = true
}
```

### Else

```base
if( condition ) {
    // TODO si condition = true
} else {
    // TODO si condition = false
}
```

### Else-if

```base
if( condition1 ) {
    // TODO si condition1 = true
} else-if( condition2 ) {
    // TODO si condition1 = false et condition2 = true
}
```

### Combinaison des trois

```base
if( condition1 ) {
    // TODO si condition1 = true;
} else-if( condition2 ) {
    // TODO si condition1 = false et condition2 = true
} else {
    // TODO si condition1 = false et condition2 = false
}
```
